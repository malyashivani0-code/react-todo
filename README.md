# React Todo List Application

## Description
This is a simple Todo List / Task Manager application built using React.
It allows users to add, view, complete, and delete tasks.
Tasks are stored in localStorage so data persists after page refresh.

## Features

- Add a todo with title and description
- View list of all todos
- Mark todos as completed or incomplete
- Delete a todo
- Prevent duplicate todos
- Data stored in browser using localStorage
- Responsive UI using Tailwind CSS
- Routing using React Router


## Tech Stack
- React
- React Router
- JavaScript (ES6)
- CSS / Tailwind CSS

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open the app in browser

## Hooks Used
- useState
- useEffect
